## ✨ Features

- 🔄 Responsive design for all devices  
- 🌙 Dark mode toggle  
- 🌍 Language toggle (English / Dutch)  
- 📜 Timeline of personal journey  
- 📞 Contact section (Email and Phone)  
- 🔼 Scroll-to-top button  
- 📱 Mobile-friendly off-canvas navigation  

## 🛠️ Built With

- **HTML5**  
- **CSS3** (Custom Properties, Flexbox, Media Queries)  
- **JavaScript (ES6+)**  
- **Font Awesome** (for icons)  
- **Unsplash** (for hero image)

## 🚀 Getting Started

To view or edit the site locally:

1. Clone this repository or download the ZIP.  
2. Open the `index.html` file directly in your browser.  
3. Edit content in `index.html`, `styles.css`, or `script.js` as needed.

## 🧩 Customization Tips

- Change colors or fonts in `styles.css`  
- Add or translate text using the `data-key` system in HTML and JavaScript  
- Replace the Unsplash hero background with your own image (in the `.hero` CSS)
